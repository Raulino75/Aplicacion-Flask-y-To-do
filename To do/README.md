# Aplicación Flask

Esta es una aplicación web simple construida con Flask.

## Requisitos

- Python 3.x
- Flask

## Instalación

1. Clona el repositorio:
   ```
   git clone https://github.com/Raulino75/Aplicacion-Flask.git
   cd Aplicacion-Flask
   ```

2. Crea un entorno virtual (opcional pero recomendado):
   ```
   python -m venv .venv
   .\.venv\Scripts\activate  # En Windows
   ```

3. Instala las dependencias:
   ```
   pip install -r requirements.txt
   ```

## Ejecución

Ejecuta la aplicación con:
```
python app.py
```

La aplicación estará disponible en `http://localhost:5000/`.

## Estructura del proyecto

- `app.py`: Archivo principal de la aplicación Flask.
- `templates/index.html`: Plantilla HTML para la página de inicio.
- `requirements.txt`: Lista de dependencias.
- `to-do.py`: (Descripción si aplica)
